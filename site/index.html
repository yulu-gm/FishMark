设计哲学

少一点打扰，
多一点专注

FishMark 围绕日常写作场景打磨：打开、编辑、保存、整理，每一步都尽量轻。

原文清楚
保存时尊重已有换行、缩进与标记，让文档在常见 Markdown 工具里继续保持清晰。
编辑自然
光标所在行显示 Markdown 原文，离开后恢复阅读排版。写作与阅读在同一页面里完成。
文件安心
打开、保存、另存为和外部变更提醒都围绕桌面文件完成，适合长期维护自己的文档库。
功能特性

常用能力，
安静放好

阅读态与源码态切换
标题、段落、列表、任务列表、引用块、代码块和分割线在非激活态呈现为阅读样式，光标移入即可编辑原文。
多标签页工作区
单窗口内同时打开多个文档，支持切换、关闭、排序和拖出成新窗口，每个标签独立管理保存状态。
新功能
可折叠大纲侧栏
基于标题层级自动生成大纲，右侧悬浮独立滚动，默认收起，点击可跳转至对应位置。
剪贴板粘贴图片
粘贴截图或剪贴板图片时自动生成资源文件，并插入相对路径 Markdown，图片在编辑区即时可见。
自动保存与变更提醒
停止输入后自动保存，窗口失焦时也会保存。外部程序修改同一文件时给出提示，方便选择重载或另存。
主题与偏好设置
颜色模式、文档字体、字号和自动保存延迟可在设置里调整，写作区域可以按自己的习惯收拾干净。
查找替换与 HTML 导出
支持当前文档查找替换、匹配高亮和替换后撤销；完成的文稿可以导出为 HTML，保留当前主题下的阅读效果。
