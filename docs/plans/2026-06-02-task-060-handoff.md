Task: TASK-060 整篇源码模式

Changed:
- Added `EditorViewMode = "wysiwym" | "source"` to editor-core and exposed a CodeMirror `StateField` / `StateEffect` update path for switching modes without rebuilding `EditorState`.
- Gated editor-core Markdown preview decorations in source mode so only physical line decorations remain; inactive previews, inline marker hiding, widgets, table widgets, task checkbox widgets, image previews, quote rails, code highlighting and thematic-break previews are omitted.
- Disabled hidden-marker / structural navigation normalization and block pointer interaction resolution while source mode is active.
- Threaded `viewMode` through `CodeEditorController`, `CodeEditorView`, `App.tsx`, and `WorkspaceShell`.
- Added status bar `</>` toggle with `aria-pressed`, `onMouseDown.preventDefault()`, and window-scoped renderer state.
- Added source-mode tests for editor-core decorations, CodeEditorController toggling, CodeEditorView prop updates, WorkspaceShell toggle UI, and autosave-sensitive focused coverage.
- Added TC-060-SOURCE manual acceptance steps.

Manual acceptance draft:
1. Open a Markdown file with heading, link, image, table, task list, blockquote, code fence, thematic break and inline formatting.
2. Click `</>` in the status bar.
3. Confirm raw Markdown is visible for every structure and no preview widget remains.
4. Edit text in source mode, undo, and confirm selection/content remain stable.
5. Switch to another tab in the same window and confirm source mode remains active.
6. Click `</>` again and confirm WYSIWYM preview returns.
7. Save/export HTML and confirm source mode does not change Markdown content or dirty state.

Known risks or not-done items:
- Status bar remains hidden in the existing reading-mode chrome collapse, so users enter editing chrome before using the status-bar source toggle. The toggle itself does not steal focus once visible.
- Source-mode scroll preservation is indirectly covered by avoiding `EditorState` rebuild, but no visual scroll probe was added in this pass.
- TASK-045/046/051 preview implementations remain TODO; source mode now provides the expected gate for those future previews.
