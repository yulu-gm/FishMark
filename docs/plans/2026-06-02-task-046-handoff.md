# TASK-046 Handoff

## Scope

TASK-046 implements first-slice Markdown math support:

- inline math: `$...$`
- block math: `$$...$$`
- parser-owned `inlineMath` / `blockMath` ranges
- editor inactive preview with lazy KaTeX widget rendering
- active source restore and TASK-060 source-mode preview gate
- HTML export through KaTeX `renderToString({ output: "mathml" })`
- source fallback for ambiguous or invalid syntax

## Syntax Rules

- `$x^2$` is inline math.
- `$$\n...\n$$` is block math.
- Unclosed inline `$` remains text.
- Currency-looking text such as `Price is $5 and $6` remains text.
- Escaped `\$` remains text.
- Code span and code fence contents do not create math nodes.
- Unclosed block math is preserved as a recoverable `blockMath` block and rendered/exported as source.

## Rendering Strategy

- `markdown-engine` owns ranges and AST; renderer/export do not rescan Markdown for math ranges.
- editor-core uses `MathPreviewWidget` with dynamic `import("./katex-preview-renderer")`; the lazy adapter imports both `katex/dist/katex.min.css` and `katex` so editor previews load KaTeX CSS / font assets without entering the initial editor closure.
- HTML export stays in the existing lazy `export-html` chunk and calls `katex.renderToString` with MathML output, then wraps display math in `.katex-display`.
- Export uses a minimal inline CSS layer and MathML so the saved HTML remains a single offline-readable file without external KaTeX font URLs.
- `perf:bundle` now requires a lazy `katex` chunk and forbids `katex` in the initial source group.

## Manual Check

Use a document like:

~~~markdown
Inline $x^2$ and escaped \$x$.

$$
a + b
$$

Price is $5 and $6.

`$not_math$`

```tex
$not_math_either$
```
~~~

Expected:

- inactive WYSIWYM shows math previews for `$x^2$` and `$$...$$`
- clicking formula lines restores source text
- source mode shows raw Markdown and no math preview widgets
- currency / escaped / code math-looking text stays source text
- exported HTML contains KaTeX MathML markup and inline CSS without external font URLs
